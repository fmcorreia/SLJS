This open-source project aims to provide a mechanism for converting Silverlight applications into HTML and JavaScript, partially because it might be of some use to people, but mostly because it is a fun project to work on.

In its infancy still, it is hoped that by openly posting progress and code that other developers will help form a community - all input is welcomed and all contributors will receive public recognition for their work.

NB: These have been tested in Chrome and Firefox, mileage may vary.

Demonstrations

Hello World (Silverlight | HTML)

Demonstrating the proof of concept, a button and a label with dependency properties, bound-css-styling and UI events.

Calculator (Silverlight | HTML)

Slightly more advanced, demonstrating that we have some of the basic layout concepts, event routing and the ability to update renderable components based on the properties of the control.
