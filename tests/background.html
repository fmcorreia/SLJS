Like all good projects, SLJS was started after a discussion in a bar during a conference.

The timeline looks something like this:

  • 2011-06-08: Discussion in bar at NDC2011
  • 2011-06-11: First POC completed, failure :-(
  • 2011-06-13: Back home, second POC success :-)
  • 2011-06-20: Re-written, several technologies played with
  • 2011-06-23: Technology chosen/structure-decided
  • 2011-06-28: First demos finished and supported
  • 2011-06-29: Time to ask for the experts to come in!

Several templating mechanisms have been toyed with, two IL->JS conversion libraries have been evaluated, and various levels of trying to convert some of the core Silverlight code across have been attempted, but in the end it was decided that it would be easier to re-write most of the framework in pure Javascript from the ground up, and simply to focus on the important bits first.

Hopefully this should shed some perspective on the project, it took a month to get anything translated from meatspace into a github repository!

With any luck, these technology decisions will last us a month or so before having to re-evaluate.
